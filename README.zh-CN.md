# Finance-Pro
一个基于网页的全球金融平台，提供纳斯达克100（NASDAQ-100）、标普500（S&P 500）、美元指数等的实数据、模拟交易和由 AI 驱动的分析。  
由 KIMI K2 LLM 辅助开发；原始数据经 Yahoo Finance 授权。
