# Finance-Pro
[简体中文版本](./README.zh-CN.md)
A web-based global finance platform that delivers real-time dashboards, mock-trading, and AI-driven analysis for the NASDAQ-100, S&P 500, US Dollar Index and more.  
Assisted by KIMI K2 LLM; raw data licensed from Yahoo Finance.
