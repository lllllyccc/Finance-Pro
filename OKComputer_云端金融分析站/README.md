# FinancePro - 金融投资分析平台

## 项目概述

FinancePro是一个专业的实时金融数据分析平台，基于Cloudflare免费版构建，提供美元指数、新兴市场ETF、纳指100、标普500等关键金融指数的实时监控和智能投资建议。

## 核心功能

### 🎯 实时数据监控
- **美元指数 (DXY)** - 实时价格、涨跌幅、成交量监控
- **主要ETF跟踪** - QQQ（纳指100）、SPY（标普500）、EEM/VWO（新兴市场ETF）
- **技术指标** - RSI、MACD、移动平均线等实时计算
- **多时间框架** - 支持1天、1周、1月、3月、1年数据查看

### 📊 智能投资建议
- **风险评估问卷** - 个性化投资偏好分析
- **资产配置建议** - 基于风险偏好的智能配置
- **市场分析报告** - 多维度市场趋势分析
- **投资策略生成** - 量化分析驱动的投资建议

### 💼 投资组合管理
- **虚拟投资组合** - 支持多种资产的组合构建
- **实时盈亏计算** - 动态更新投资组合价值
- **风险分散分析** - 投资组合风险度量和优化
- **历史回测功能** - 1个月至5年的历史表现分析

### 🌐 多市场覆盖
- **美股市场** - 道指、纳指、标普500综合分析
- **港股市场** - 恒生指数、国企指数分析
- **国内股市** - 沪深300、中证500实时分析
- **外汇市场** - 美元指数、主要货币对分析
- **基金市场** - 各类ETF表现对比分析

2. **创建Pages项目** - 选择"Upload assets"选项
3. **上传项目文件** - 打包ZIP文件上传
4. **配置项目设置** - 设置项目名称和部署参数
5. **获取访问URL** - 部署完成后获得唯一访问地址
